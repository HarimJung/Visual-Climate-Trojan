# 콘텐츠 전략: 정보성 vs 제품

## 원칙

같은 스킬, 같은 데이터. 정보성에는 "과정 + 일부"를, 제품에는 "전체 완성본"을 넣는다.

## 정보성 콘텐츠 (무료, LinkedIn 배포)

| 콘텐츠 | 만드는 스킬 | LinkedIn 형태 | 목적 |
|--------|-----------|-------------|------|
| 소스 비교표 | source-comparator | 캐러셀 4장 | Save + 팔로우 |
| 용어 정의 + 체크리스트 | indicator-definer | 캐러셀 5장 | Save + 전문성 |
| 데이터 품질 Before/After | data-quality-checker | 이미지 1장 + 텍스트 | 교육 |
| Country Brief Key Metrics만 | country-brief (일부) | 이미지 1장 | 제품 맛보기 |
| Tableau 차트 해설 | tableau-spec-writer + Tableau | 이미지 1장 + 텍스트 | 시각화 역량 |
| NDC Gap 한 줄 요약 | ndc-gap-brief (일부) | 텍스트 포스트 | 시의성 |

## 제품 (유료 또는 리드 생성)

| 제품 | 만드는 스킬 | 전달 형태 | 수익화 |
|------|-----------|----------|--------|
| Country Climate Brief 전문 | country-brief | PDF (이메일 수집) | 리드 |
| Comparison Brief | comparison-brief | PDF (이메일 수집) | 리드 |
| NDC Gap Brief | ndc-gap-brief | PDF ($49/국가) | 직접 수익 |
| 24주 브리프 묶음 | 위 3개 반복 | PDF 번들 | 직접 수익 |
| Tableau 대시보드 | tableau-spec-writer + Tableau | Tableau Public + 임베드 | 트래픽 |
| 이 시스템 구축 강의 | 전체 과정 문서화 | 영상/PDF | 직접 수익 |
| 맞춤 컨설팅 | 전체 스킬 활용 | 보고서 + 대시보드 | $10K+ |
