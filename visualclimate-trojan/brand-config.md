# VisualClimate — Brand Configuration

## Colors

| Name | Hex | Use |
|------|-----|-----|
| Primary | #0066FF | Links, CTAs, main accent |
| Changer (Good) | #00A67E | Positive metrics, On Track |
| Starter (Warning) | #F59E0B | Mixed metrics, Narrow Gap |
| Talker (Bad) | #E5484D | Negative metrics, Significant Gap |
| Critical | #7F1D1D | Worst case, Critical Gap |
| Background | #FFFFFF | All backgrounds |
| Section BG | #F8F9FA | Alternating sections |
| Text Primary | #1A1A2E | Headings, body text |
| Text Secondary | #4A4A6A | Subtitles, labels |
| Text Muted | #8888A0 | Captions, footnotes |
| Border | #E5E7EB | Dividers, card borders |
| Data Missing | #CCCCCC | [DATA MISSING] indicators |

## Typography

| Use | Font | Weight | Size |
|-----|------|--------|------|
| Headings | Inter | Bold (700) | 14-24pt |
| Body | Inter | Regular (400) | 11-14pt |
| Data/Code | JetBrains Mono | Regular | 10-12pt |
| Captions | Inter | Regular | 9-10pt |

## Logo

- Position: bottom-right on all outputs
- Size: 80 × 20 px on carousel slides
- Text-only fallback: "VisualClimate" in Inter Bold #0066FF

## Report Header

Every report starts with:
```
{Report Title}
Data vintage: {year} | Generated: {date} | Engine: VisualClimate Trojan v1.0
```

## Carousel Slide Rules

- 1080 × 1080 px
- White background (#FFFFFF)
- Max 2 colors per slide (Primary + one accent)
- Source citation: bottom-left, 9pt, #8888A0
- Logo: bottom-right

## Prohibited

- Dark backgrounds (bg-slate-900, #0a0a1a, etc.)
- More than 3 accent colors per piece
- Stock photos
- Emojis in reports (ok in LinkedIn captions, max 2)
